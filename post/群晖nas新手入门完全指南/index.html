群晖 Synology NAS 新手入门完全指南

从拆箱到进阶，一篇搞定群晖 NAS 的安装、配置、安全设置、核心功能和远程访问

Wednesday, March 25, 2026 2 minutes read

如果你刚入手了一台群晖 NAS，或者正在考虑入手，这篇文章就是为你准备的。从硬件安装到进阶玩法，一步步带你上手。

📦 一、收到设备后做什么

1.1 硬件检查

确认包装内配件齐全：主机、电源线、网线、硬盘笼螺丝。

群晖主机不含硬盘，需自购。推荐：

  • 希捷酷狼 IronWolf — NAS 专用盘，CMR 技术
  • 西数红盘 Plus — 同样是 NAS 专用

1.2 安装硬盘

  1. 关机并拔掉电源线
  2. 抽出硬盘架，固定硬盘（3.5 寸用侧面螺丝，2.5 寸用底部螺丝）
  3. 推回硬盘架，接上电源线和网线
  4. 开机，等待指示灯变为绿色常亮

1.3 初始化设置

  1. 在电脑访问 find.synology.com 或 http://NAS内网IP:5000
  2. 按向导完成 DSM 系统安装
  3. 设置管理员账户和强密码
  4. 选择"自动更新"保持系统最新

💾 二、存储池规划（SHR 新手推荐）

什么是 SHR？

SHR（Synology Hybrid RAID）是群晖自研的智能 RAID，比传统 RAID 更灵活，硬盘利用率更高。

硬盘组合建议

硬盘数 推荐模式 说明
1 块 Basic 无冗余，但灵活
2 块 SHR-1（推荐） 1 块硬盘容错
3 块+ SHR-1 1 块硬盘容错
4 块+ SHR-2 2 块硬盘容错（安全性更高）

⚠️ 新手千万别组 RAID0！ 一块硬盘损坏 = 所有数据全丢。

创建存储池步骤

  1. 进入存储管理器 → 存储池 → 新增
  2. 选择 SHR（自动模式）
  3. 选择要加入的硬盘
  4. 完成初始化

🔐 三、安全设置（必做！）

3.1 开启防火墙

控制面板 → 安全性 → 防火墙 → 编辑规则

只开放必要端口，禁用 DMZ。

3.2 禁用 Admin 账户

控制面板 → 用户与群组 → 本地用户

新建日常使用账户并赋予管理员权限，禁用默认 admin 账户。

3.3 开启两步验证（2FA）

控制面板 → 用户与群组 → 账户 → 两步验证

下载 Google Authenticator 或 Authy App 绑定。

📁 四、核心功能使用

4.1 File Station（文件管理器）

网页端直接管理文件，支持拖拽上传/下载，可创建共享链接。

4.2 Drive（私有云盘）⭐ 强烈推荐

  • 同步文件到电脑/手机（类似 OneDrive）
  • 支持多人协作编辑
  • 版本历史（误删可恢复）

客户端支持：Windows / Mac / iOS / Android

4.3 Photo Station / Moments（照片备份）

  • 手机安装 Synology Photos App
  • 开启自动备份照片到 NAS
  • 从此解放手机存储空间

4.4 Download Station（下载机）

集成 PT / BT / 磁力链下载，配合私有云盘直接管理下载文件。

4.5 Surveillance Station（监控录像）

接入摄像头后支持移动侦测、录像回放。

🐳 五、Docker 入门（进阶）

💡 等熟悉基础操作后再折腾，新手阶段可以先跳过此章节。

Docker 能做什么？

  • 跑 PT 下载工具（qBittorrent、Transmission）
  • HomeAssistant 智能家居
  • Jellyfin / Emby 影视库
  • Nginx 反向代理
  • 各种自动化脚本工具

基础概念

概念 解释
镜像（Image） 应用的模板
容器（Container） 运行起来的实例
端口映射 容器端口 → 主机端口
卷（Volume） 容器数据的持久化存储

推荐新手先安装

  1. 安装 Container Manager 套件
  2. 安装 Portainer（可视化管理 Docker）

🌐 六、远程访问设置

6.1 内网访问

直接用 NAS 的内网 IP 访问，如 http://192.168.1.100:5000

6.2 外网访问 — QuickConnect（适合新手）

  1. 控制面板 → 外部访问 → QuickConnect
  2. 注册 Synology 账号
  3. 获取专属 QuickConnect ID
  4. 之后用 https://你的ID.quickconnect.to 随时随地访问
  • ✅ 优点：无需设置路由器端口转发
  • ⚠️ 缺点：速度比内网稍慢

6.3 外网访问 — DDNS + 端口转发（更专业）

申请公网 IP → 设置路由器端口转发 → 使用群晖内置 DDNS 或花生壳服务。

🛡️ 七、数据备份建议

3-2-1 备份原则

  • 3 份数据副本
  • 2 种不同存储介质（如 NAS + 外接硬盘）
  • 1 份异地备份（云端或另一台 NAS）

群晖内置备份工具

  • Hyper Backup：备份到本地/外接硬盘/另一台 NAS/云端
  • Snapshot Replication：快照备份（需配合 Btrfs 文件系统）

📚 八、新手学习路径

时间 任务
第 1 天 完成初始化、存储池创建、基础安全设置
第 2 天 体验 File Station、Drive 文件同步
第 3 天 Photo Station 照片备份
第 4 天 开启 QuickConnect 远程访问
第 5 天+ 学习 Docker，折腾更多玩法

⚠️ 常见问题

Q：硬盘发出异响怎么办？ 立刻关机备份数据，可能硬盘即将损坏。

Q：忘记管理员密码怎么办？ 重置 NAS（按住 reset 键 4 秒），但会丢失数据。

Q：NAS 能 24 小时开着吗？ 可以，群晖设计为 7×24 小时运行。注意散热和定期检查硬盘健康。

Q：能装 Windows 虚拟机吗？ 可以，安装 Virtual Machine Manager 套件，分配 CPU/内存资源即可。
